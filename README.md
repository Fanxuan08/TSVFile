# TSV Vocabulary Learner (TSV 單字背誦管理程式)

## 專案簡介

本專案為一個基於 C# Windows Forms 開發的單字記憶與 TSV 檔案讀取工具。在基礎的檔案讀取與清單顯示之上，大幅強化了「使用者體驗 (UX)」與「資料持久化 (Data Persistence)」，將原本靜態的資料展示器，進化為高效率的單字背誦神器。

### 基礎功能 (核心架構)

* **TSV/TXT 檔案解析**
  * 運用標準的檔案 I/O (`File.ReadAllLines`) 讀取外部單字檔，並透過 `Split('\t')` 精準解析單字、音標、音檔路徑與解釋。
* **物件導向設計 (OOP)**
  * 將單一單字封裝為 `WordItem` 類別，並將單字群組集合管理於繼承自 `Collection<WordItem>` 的 `WordCollection` 中，使資料操作更為直觀。
* **ListView 資料綁定**
  * 使用 `ListView` 條理分明地呈現單字資訊，並使用 `BeginUpdate` 與 `EndUpdate` 優化大量資料載入時的重繪效能。

### 進階創意擴充

* **全鍵盤快捷操作 (沉浸式背誦)**
  * 為解決滑鼠與鍵盤頻繁切換的效率耗損，實作了全鍵盤支援：
    * 透過攔截 `KeyDown` 事件，使用者可直接使用 **↑ (上)** 與 **↓ (下)** 方向鍵快速切換單字。
    * 結合 `EnsureVisible()` 方法，確保使用方向鍵往下瀏覽時，畫面會自動平滑滾動，單字永遠保持在可視範圍內。
* **進度保存與自動存檔 (Data Persistence)**
  * **打勾記憶功能**：內建「已背誦」的核取方塊 (`IsMemorized`)。使用者每勾選或取消一個單字，系統會立即觸發 `ItemChecked` 事件。
  * **即時覆寫機制**：狀態改變時，程式會自動呼叫自訂的 `SaveToFile` 方法，將最新的 True/False 狀態即時寫入原檔的最後一欄。下次開啟程式時，進度絕對不會遺失。
* **向後相容的防呆解析 (Backward Compatibility)**
  * 在 `WordItem` 的建構子中加入了防呆邏輯 (`bool.TryParse`)。若讀取到舊版（沒有打勾狀態）的單字檔，系統依然能正常解析解釋欄位，並自動將新屬性預設為 `false`，完美相容各代版本的 TSV 檔案。
* **智慧化 UI 調整與檔案過濾**
  * **無障礙開檔**：擴充 `OpenFileDialog` 的 `Filter`，並將預設檢視設為所有文字檔，減少使用者尋找檔案的麻煩。
  * **友善閱讀視野**：於程式載入 (`Load`) 時，強制重新分配 `ListView` 的欄寬（例如將「解釋」欄位大幅拉寬至 500px），解決長句被截斷的問題，提升閱讀體驗。

### 1. 載入單字檔
* 點擊選單列的「開啟檔案」，系統預設會尋找 `.txt` 或 `.tsv` 格式的單字檔。
* 成功載入後，左下角狀態列會提示「X 單字已成功載入」。

  <img width="700" height="400" alt="螢幕擷取畫面 2026-05-25 154823" src="https://github.com/user-attachments/assets/4467179f-3c34-4cce-a6c8-356811fe0134" />


### 2. 高效單字背誦
* 單手操作鍵盤 **↑ (上) / ↓ (下)** 方向鍵即可流暢切換單字，畫面會自動跟隨捲動。
* 按下 **Space (空白鍵)** 即可直接勾選/取消該單字的「已背誦」狀態。

  <img width="700" height="400" alt="螢幕擷取畫面 2026-05-25 154850" src="https://github.com/user-attachments/assets/d6060566-f9c9-4fbc-8109-88b649027899" />

### 3. 進度自動存檔
* 當您勾選任何一個單字時，程式已在背景悄悄完成存檔。您可以隨時關閉程式，下次重新載入同一個檔案時，所有打勾的進度都會完美還原。
